Bus Books is the BusDK product line for auditable financial workflows over
inspectable workspace data. It connects bookkeeping UI, invoices, evidence,
reconciliation, validation, reports, deterministic formulas, PDFs, CLI, API,
and agent-facing financial tools over the same data.

Use this product when the goal is practical bookkeeping, evidence handling,
period/VAT/report workflows, or technical evaluation of a deterministic
accounting engine.

## Start Here

1. Read the [accounting workflow overview](../workflow/accounting-workflow-overview).
2. Follow the [end-to-end workflow](../workflow/) when evaluating a sample
   workspace.
3. Use [sale invoicing](../workflow/sale-invoicing), [bank import and payment

4. Read [workbook editing](../workflow/workbook-and-validated-tabular-editing)
   when evaluating Bus Formula Language and data workbench surfaces.

## Product Modules

Primary user-facing modules include [`bus books`](../modules/bus-books),
[`bus ledger`](../modules/bus-ledger), [`bus portal accounting`](../modules/bus-portal-accounting),
[`bus data`](../modules/bus-data), [`bus sheets`](../modules/bus-sheets),
[`bus files`](../modules/bus-files), [`bus attachments`](../modules/bus-attachments),
[`bus bfl`](../modules/bus-bfl), and [`bus pdf`](../modules/bus-pdf).

Accounting domain modules include accounts, balances, bank, budget, customers,
entities, invoices, journal, memo, period, reconcile, reports, validate, VAT,
vendors, assets, loans, inventory, payroll, and related workspace datasets.
