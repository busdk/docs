## Create an accounting entity

Create a new bookkeeping scope so journals and VAT never mix across companies.

Owner: [bus init](../../modules/bus-init).

Create an accounting entity when you need to keep two companies or ledgers separate in the same repository. Every bookable record must belong to exactly one entity so journals, VAT, and reports remain correct.
