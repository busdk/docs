## `doc_date` (document date)

`doc_date` is the documentâ€™s own date. Bookkeeping uses document date for period selection and searching, because using upload time as a proxy becomes unreliable when documents are received or scanned late.

An explicit document date helps evidence trails remain filterable by month and fiscal period without opening files.

Example values: `2026-02-05`, `2026-01-31`.
