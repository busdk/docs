# Minimal example layout

A minimal example layout is:

```text
my-business-books/
  README.md
  datapackage.json
  accounts/
    accounts.csv
    contacts.csv
    accounts.schema.json
    contacts.schema.json
  journal/
    journal_2026.csv
    journal_2026.schema.json
  invoices/
    sales_invoices.csv
    sales_invoice_lines.csv
    purchase_invoices.csv
    purchase_invoice_lines.csv
    sales_invoices.schema.json
    sales_invoice_lines.schema.json
    purchase_invoices.schema.json
    purchase_invoice_lines.schema.json
    pdf/
      INV-1001.pdf
  vat/
    vat_rates.csv
    vat_return_2026Q1.csv
    vat_rates.schema.json
    vat_return_2026Q1.schema.json
  budget/
    budgets.csv
    budgets.schema.json
```
