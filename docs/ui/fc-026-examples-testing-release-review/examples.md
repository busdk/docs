- [Assistant panel](../fc-009-assistant-workbench-shell/assistant-panel) shows assistant shell and AI
  panel markup.
- [Evidence preview](../fc-019-evidence-previews/evidence-preview) shows
  evidence preview and event markup.
- [Portal modules](../fc-025-product-module-integration/portal-modules) shows portal module mounting and

- [Testing UI apps](../fc-026-examples-testing-release-review/testing-guide) shows fixture and renderer-test usage.
