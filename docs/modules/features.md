## BusDK module capabilities

Use this page as a quick end-user map of what each module is for. For command syntax, flags, and files, open the module page.

| Module | End-user focus |
| --- | --- |
| [bus](./bus) | Single command entrypoint and `.bus` file execution |
| [bus-init](./bus-init) | Initialize workspace datasets |
| [bus-config](./bus-config) | Workspace accounting entity configuration |
| [bus-data](./bus-data) | Dataset, schema, and package operations |
| [bus-api](./bus-api) | Local HTTP API over workspace data |
| [bus-sheets](./bus-sheets) | Spreadsheet-style workspace UI |
| [bus-books](./bus-books) | Bookkeeping UI workflows |
| [bus-accounts](./bus-accounts) | Chart of accounts management |
| [bus-entities](./bus-entities) | Counterparty/entity master data |
| [bus-period](./bus-period) | Accounting period lifecycle |
| [bus-attachments](./bus-attachments) | Evidence attachment registration and linking |
| [bus-invoices](./bus-invoices) | Sales and purchase invoice workflows |
| [bus-journal](./bus-journal) | Double-entry postings and balances |
| [bus-bank](./bus-bank) | Bank account and transaction ingestion |
| [bus-reconcile](./bus-reconcile) | Matching and allocation across bank/invoice/journal |
| [bus-assets](./bus-assets) | Fixed-asset lifecycle and postings |
| [bus-loans](./bus-loans) | Loan register, events, schedules, postings |
| [bus-inventory](./bus-inventory) | Inventory items, movements, valuation |
| [bus-payroll](./bus-payroll) | Payroll dataset validation/export |
| [bus-budget](./bus-budget) | Budget dataset management and reporting |
| [bus-reports](./bus-reports) | Financial and statutory reports |
| [bus-validate](./bus-validate) | Workspace-wide validation and diagnostics |
| [bus-vat](./bus-vat) | VAT validation, reporting, and export |
| [bus-pdf](./bus-pdf) | Deterministic PDF rendering |
| [bus-filing](./bus-filing) | Filing target dispatch |
| [bus-filing-prh](./bus-filing-prh) | PRH-specific filing bundle workflows |
| [bus-filing-vero](./bus-filing-vero) | Vero-specific filing bundle workflows |
| [bus-replay](./bus-replay) | Export/apply/replay workspace operations |
| [bus-dev](./bus-dev) | Developer workflow automation |
| [bus-agent](./bus-agent) | Agent runtime detection/render/run utilities |
| [bus-run](./bus-run) | Prompt/script/pipeline execution |
| [bus-preferences](./bus-preferences) | User preference storage |
| [bus-secrets](./bus-secrets) | Secret reference management |
| [bus-shell](./bus-shell) | Interactive or one-shot command shell |
| [bus-status](./bus-status) | Workspace readiness/status checks |
| [bus-update](./bus-update) | Update checks and policy enforcement |
